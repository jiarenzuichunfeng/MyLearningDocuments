# Kotlin 语言基础

* 数据类型
  * 字符型 Char 2字节
    * 不能是用隐形转换来进行计算，必须强制转换
  * 字符串 String
  * 数字类型
    * 整形
      * 字节Byte 1字节
      * 短整型Short 2字节
      * 整形 Int 4字节
      * 长整型 Long 8字节
      * 无符号字节UByte
      * 无短整型UShort 2字节
      * 无整形 UInt 4字节
      * 无长整型 ULong 8字节
    * 浮点型
      * 单精度浮点型 Float 4字节 保留6~7位小数
      * 双精度浮点型 Double 8字节 保留15~16位小数
  * 布尔值 Boolean 值只能是 true或false
* 运算符
  * 数学运算符
    * +
    * -
    * *
    * /
    * %
  * 逻辑运算符
    * 
* 格式化输出
  * $ 变量名 如果是表达式 ${表达式}
* 注释
  * 单行注释 //
  * 多行注释 /**/
  * 文档注释/***/
    * 参数
      * @paran 形参
      * @return 返回值
      * @property 描述类的属性（适用于主构造函数参数、类属性）
      * @throws 描述函数可能抛出的异常（Kotlin 无检查异常，仅用于说明）
      * @see 引用其他相关的类 / 函数 / 属性（支持链接：`@see [sum]` 或 `@see java.lang.Math`）
      * @sample 代码示例
      * @author 标注作者
      * @since 标注代码新增的版本
* 变量
  * 声明变量
    * var 变量名：数据类型 = 值
* 常量
  * 声明常量
    * val 常量名 ： 数据类型 = 值
* 函数
  * 定义函数
    * fun 函数名 （形参： 数据类型 = 默认值）： 返回值类型 {函数体}
  * 调用函数 函数名 （实参）
