# 基本数据类型与包装数据类型

* 基本类型不具有对象的性质，为了让基本数据类型也具有对象的特征，就出现了包装数据类型

* 集合框架里需要存储对象，不能存储基本数据类型，所以需要存储包装类型

* java里面的包装数据类型、

  ```
  基本类型     包装器类型
  boolean;    Boolean;
  char;				Character;
  int;  			Integer;
  byte;				Byte;
  short;			Short;
  long;				Long;
  float;			Float;
  double;			Double;
  ```

* 基本数据类型与包装数据类型相互转换

  * 基本类型转包装类型，需要new
  * 包装类型转基本类型，直接调方法

* 基本数据类型与包装数据类型的区别

  * 基本数据类型不用new，包装类型需要用new关键字来在堆张分配存储空间
  * 存储方式及位置不同，基本类型是直接将变量值存储在栈中，包装类型是将对象放在堆中，然后通过引用来使用
  * 初始值不同，基本类型的初始值一个确定的值，包装类型的初始值为null
