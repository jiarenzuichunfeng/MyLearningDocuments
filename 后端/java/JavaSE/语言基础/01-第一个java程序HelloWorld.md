# HelloWord

1. 随便创建一个文件夹用来存放代码
2. 新建一个java文件
   * 后缀名为,java
   * Hello.java
3. 编写代码

```java
public class Hello {
	public static void main (String[] args){
		System.out.print("Hello Java");
	}
}
```
4. 编译 javac java文件，会生成一个class文件
5. 运行class文件 java class文件







# 可能出现的问题

1. 每个单词的大小不能出现问题，**Java是大小写敏感的**
1. 尽量使用英文
1. 文件名 和 类名必须保证一致 并且首字母大写
