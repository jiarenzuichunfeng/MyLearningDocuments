# Dos命令

## 常用的Dos命令

``` bash
#查看当前目录下的所有文件 dir
	#常用参数 
		#/p：当目录内容繁多时，实现逐屏展示，便于浏览
		#/w：以简洁的宽格式呈现，仅列出文件名与目录名，每行多个显示
#创建目录 md 目录名
#删除目录 rd 目录名
#切换目录 cd空格目录路径
#切换盘符 cd空格:盘符路径
#返回上一级 cd..
#文件复制 copy 文件名 新文件名
	#如果目录则需要加上路径
#文件移动 move 文件名 文件地址
#文件删除 del 文件名
#格式化磁盘 format
#检查磁盘状态 chkdsk
#清理屏幕 cls
#退出终端 exit
#systeminfo：全面展示计算机的详细系统信息，涵盖操作系统版本、BIOS 版本、处理器型号等诸多内容。
#ipconfig：呈现当前 TCP/IP 网络配置详情，包括 IP 地址、子网掩码、默认网关等。
```
